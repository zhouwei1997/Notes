## 常用通配符

| 通配符                       | 含义                                       |
| ---------------------------- | ------------------------------------------ |
| *                            | 匹配0到多个任意字符                        |
| ?                            | 匹配任意单个字符                           |
| [list]                       | 匹配[list]中的任意单个字符，或一组单个字符 |
| [!list]                      | 匹配除了[list]中的任意单个字符             |
| {string1,string2,...stringn} | 匹配string1，string2或更多字符串           |
